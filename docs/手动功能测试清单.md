# 🧪 Wiza潜在客户搜索工具 - 手动功能测试清单

## 测试前准备

### 环境要求
- ✅ Windows 10/11 系统
- ✅ Node.js v18+ 已安装
- ✅ 有效的Wiza API密钥 (用于完整测试)
- ✅ 网络连接正常

### 启动应用
```bash
# 开发模式启动
npm run dev

# 或者启动Electron应用
npm run electron:dev
```
